SCRAMBLED WORDS


SCRAMBLED words edition. In this game, all players get the same random set of letters.

They then drag and drop the letters to build a word. If the word is a valid dictionary word (text wordlist of 100,000+ English Words), they get 3 points for each letter in the word. Words already found using the current scrambled letters don't count.

Each time a word is found, the player name and his/her score is shown.

A high score is kept.

Uses the graphics handling library PIXI js.

Uses the sound library ion sound.

Uses Node.js and Socket.io for multiplayer capability in client/server framework.
